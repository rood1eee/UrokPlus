# База данных UrokPlus (PostgreSQL)

## Создание БД и применение схемы

```bash
# Создать БД (если ещё нет)
createdb urokplus

# Применить скрипт
psql -U postgres -d urokplus -f init.sql
```

Или через psql вручную:

```bash
psql -U postgres -d urokplus
\i init.sql
```

## Таблицы

| Таблица   | Описание |
|-----------|----------|
| `users`  | Пользователи (логин, хэш пароля, роль: STUDENT/PARENT/TEACHER) |
| `profile`| Профиль пользователя (имя, школа, класс, аватар), связь по `user_id` |
| `messages` | Сообщения в чатах (`chat_id`, текст, тип TEXT/IMAGE/VOICE) |
| `grades` | Оценки и уведомления (ученик, предмет, оценка, тип работы) |
| `lessons` | Расписание уроков по дням недели (опционально) |

Скрипт соответствует моделям приложения в `AuthModels.kt` и таблицам в `AuthRepository.kt`.
