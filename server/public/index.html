UrokPlus API

REST API для приложения UrokPlus. Соответствует данным: пользователи, профиль, чаты/сообщения, оценки. База данных — PostgreSQL.

Проверка работы

GET
/api/health

Ответ: { "ok": true, "database": "connected" }

Авторизация

POST
/api/auth/register

Тело (JSON): login, password, confirm, role (STUDENT | PARENT | TEACHER)

{
  "login": "ivan",
  "password": "secret123",
  "confirm": "secret123",
  "role": "STUDENT"
}

Успех: { "success": true, "user": { "id", "login", "role" } }

POST
/api/auth/login

Тело: login, password

Успех: { "success": true, "user": { "id", "login", "role" } }

Профиль

GET
/api/profile

Заголовок X-User-Id или query ?userId=1 — id пользователя.

Ответ: { "name", "school", "grade", "avatarUrl" }

PUT
/api/profile

Тело: name, school, grade, avatarUrl (опционально userId)

Сообщения чата

GET
/api/chats/:chatId/messages

Список сообщений чата. chatId — строка (например, имя чата).

Ответ: массив { "id", "chatId", "text", "isMe", "type", "timestamp" }, type: TEXT | IMAGE | VOICE

POST
/api/chats/:chatId/messages

Тело: text, isMe (bool), type (TEXT|IMAGE|VOICE), timestamp (опционально)

Оценки

GET
/api/grades

Список событий оценок. Ответ: массив { "id", "studentName", "subject", "grade", "workType", "timestamp", "isRead" }

POST
/api/grades

Тело: studentName, subject, grade, workType, timestamp (опционально)

Расписание (опционально)

GET
/api/lessons?day=1

day — день недели 1–7 (1 = понедельник). Без параметра — текущий день.

Ответ: массив уроков { "id", "name", "time", "homework", "teacher", "gradeClass" }

База данных инициализируется скриптом database/init.sql. Подключение задаётся переменной DATABASE_URL.
